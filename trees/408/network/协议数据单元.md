
---
title: 协议数据单元
term: Protocol Data Unit
---

协议数据单元是指在某一层协议中传输的数据单元. 每一层在接收来自上一层的数据时, 都会在其头部添加控制信息也即 "首部" Header, 然后将其当作一个新的 [PDU](./协议数据单元.md) 交给下一层处理. 这就是所谓的 "封装" 过程. 

| OSI 层级 | PDU | PDU 译名 |
| - | - | - |
| 7 [应用层][application] | Data | 数据 |
| 6 [表示层][presentation] |  | 数据 |
| 5 [会话层][session] | | 数据 |
| 4 [传输层][transport] | UDP Datagram <br> TCP Segment | UDP 数据报 <br> TCP 数据段 / 分段 |
| 3 [网络层][network] | IP Packet | IP 分组 / 数据包 / 报文 |
| 7 [数据链路层][data-link] | Frame | 帧 |
| 7 [物理层][physical] | Bit | 比特 |
